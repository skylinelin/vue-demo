## Node.js的安装
node安装完成后自带npm
### 切换npm镜像源
首先要安装切换工具 nrm
```
# 安装工具
npm install nrm -g

# 查看镜像源
nrm ls

# 切换镜像源
nrm use taobao

# 测试速度
nrm test npm
```
安装完成后重启一下电脑

---
## VUE的安装
```
# 进入指定目录
cd 目录名称

# 初始化
npm init -y

# 安装 vue
npm install vue --save

```
